Auto-coding

The Problem

Manual medical coding was:

  • Time-consuming and repetitive
  • Prone to inconsistency across studies
  • Difficult to scale efficiently
  • Creating review friction for data teams

Auto-coding

The Solution

From 4.90

  • WHODrug Koda integration enabled in Viedoc Admin
  • AI-generated coding suggestions inside Viedoc Coder
  • One-click application of suggested codes
  • ATC codes displayed alongside drug names

From 4.91

  • Coding operation column in exports (manual vs automatic)
  • Dictionary version visibility in Coder
  • Updated Medical Coding report templates aligned to export structure

Auto-coding

The Value

  • Increased coding efficiency
  • Improved consistency across coders
  • Reduced manual effort
  • Preserved full audit transparency
  • Higher confidence in downstream exports

Repeating Item Groups

The Problem

  • Repetitive data structures required workarounds
  • Lab analytes and similar data required repeating forms or events
  • Configuration was time-consuming and error-prone
  • Data collection felt unintuitive for site staff
  • Review often required manual transposition outside the system

Repeating Item Groups

The Solution

Native repeating item group support across the Viedoc Suite

Released in 4.92 (Feb 2026)

  • Configurable repeating item groups in Designer
  • Dynamic or fixed repeat structures in Clinic
  • Full support in Export, Import, APIs
  • Fully compliant with CDISC ODM
  • Up to 50 repeats per item group
  • Visibility conditions & validation fully supported
  • ItemGroupRepeatKey for precise targeting
  • New export formats for structured repeat handling

Repeating Item Groups

The Value

  • Less manual configuration effort
  • More intuitive data entry for sites
  • Structured exports aligned with real data models
  • Reduced need for data reshaping outside Viedoc
  • Stronger foundation for advanced analytics & integrations
